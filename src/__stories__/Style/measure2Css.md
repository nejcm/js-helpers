
# measure2Css

<p>
  Returns a css value from value.
</p>

### Usage

```js

import {measure2Css} from '@nejcm/js-helpers';

const measure = 10;
const result = measure2Css(measure);  // 10px

```
<br />

#### With string measure value:

```js

import {measure2Css} from '@nejcm/js-helpers';

const measure = '3rem';
const result = measure2Css(measure);  // 3rem

```
